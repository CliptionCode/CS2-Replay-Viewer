# CS2 Replay Viewer v0.2.5

Version 0.2.5 makes each player's economy visible directly in the team rosters throughout replay playback.

## Change Overview

- 🚀 **New Features:** Current player money in the CT and T rosters
- 🔧 **Adjusted Features:** Right-aligned money values for a cleaner, consistently spaced roster
- 📦 **Version and Documentation:** Version 0.2.5 with updated player-money documentation

## 🚀 New Features

### Player Money in Team Rosters

- Every CT and T roster button now shows the player's money at the current replay moment.
- Money appears in green while player names retain their existing team and dead-player colors.
- Reparse older demos to populate the new money data.

## 🔧 Adjusted Features

### Roster Money Alignment

- Money values are aligned to the right edge of each player button for easier comparison.
- Long player names shorten cleanly without hiding the money amount.

## 📦 Version and Documentation

- Increased the application version from `0.2.4` to `0.2.5`.
- Updated the feature guide and implementation context for player money in the team rosters.
